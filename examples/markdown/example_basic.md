# Basic Markdown Example

This file demonstrates the basic Markdown features supported by FlexiSoftMdReader.

## Headings

This is a level 2 heading. You can also use level 1 (`#`) and level 3 (`###`).

### Level 3 Heading

Deeper headings are also supported but may look similar depending on styling.

## Paragraphs and Line Breaks

This is a paragraph. Paragraphs are separated by blank lines.

This is another paragraph. Line breaks within a paragraph are preserved.

## Bullet Lists

- Item 1
- Item 2
- Item 3

With sub-items:
- Item 1
- Item 2
  - Sub-item 2.1
  - Sub-item 2.2
- Item 3

## Numbered Lists

1. First item
2. Second item
3. Third item

With mixed items:
1. Introduction
2. Main section
   - Detail A
   - Detail B
3. Conclusion

## Text Formatting

This text has **bold formatting** using double asterisks.

This text has *italic formatting* using single asterisks.

You can also use ***bold and italic*** together.

## Inline Code

Use backticks for inline code: `FlexiSoftMdReader.exe "file.md"`

Multiple code examples: `function()`, `variable_name`, `system.call()`

## Code Blocks

Use triple backticks to create code blocks:

```cpp
#include <windows.h>

int main() {
    MessageBoxW(NULL, L"Hello", L"FlexiSoftMdReader", MB_OK);
    return 0;
}
```

Another example in pseudo-code:

```
loop through items:
    process item
    display result
end loop
```

## Summary

FlexiSoftMdReader handles these basic Markdown elements effectively, making it suitable for displaying simple documentation and user guides.

For more complex features like tables and images, see the other example files.
