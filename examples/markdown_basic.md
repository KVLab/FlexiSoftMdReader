# Markdown Basics

Example demonstrating basic Markdown syntax supported by FlexiSoftMdReader.

## Headings

This is a level 2 heading.

### Level 3 Heading

Supported levels: 1, 2, 3

## Paragraphs

Paragraphs are separated by blank lines.

This is another paragraph.

## Bullet Lists

1. First

Use backticks: `variable_name` or `function()`

This is **bold** and this is *italic*.

FlexiSoftMdReader handles these basic elements effectively for documentation display.
