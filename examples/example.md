## Code Example

You can display code blocks with syntax highlighting:

```cpp
#include <windows.h>

int main() {
    MessageBox(NULL, L"Hello, Windows!", L"FlexiSoftMdReader", MB_OK);
    return 0;
}
```

Inline code is also supported: use `FlexiSoftMdReader.exe` to run the viewer.
