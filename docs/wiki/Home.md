Welcome to the FlexiSoftMdReader wiki.

FlexiSoftMdReader is a lightweight standalone Markdown viewer for Windows, designed for compatibility with older systems including Windows XP.

## Project Overview

- **Type**: Standalone Win32 application
- **Platform**: Windows, including Windows XP (SP3+)
- **Purpose**: Display Markdown documentation in a simple viewer
- **Dependencies**: None (static runtime, no .NET, no WebView2)

## Key Features

- ✓ Windows XP compatible (subsystem 5.01)
- ✓ Read-only Markdown viewer
- ✓ Custom font support (TTF or system fonts)
- ✓ Relative image path support
- ✓ UTF-8 encoding support
- ✓ Standalone executable
